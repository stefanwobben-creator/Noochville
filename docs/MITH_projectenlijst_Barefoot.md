# MITH 2026 "Biobased Noochie Barefoot" — objective, 48 projecten, rollen, uren

Klaar om als projectenbord in te richten. Bron: projectplan + urenstaat + resultatenvragen.
Projectvenster: 2026-05-01 t/m 2027-04-30. Totaal 500 uur.

## Model
- **Objective (moederdoel):** MITH-rapport "Biobased Noochie Barefoot" (resultatenverslag).
  DoD = de 7 secties uit `vragen t.b.v. resultatenverslag.docx`. Coördinatie-accountability
  ("het rapport wordt opgeleverd") bij **Strategic Lead & Founder Steward**.
- **Laag:** Objective → 5 werkpakketten (mijlpaal + venster + urenbudget) → projecten (elk een
  deliverable met eigen uren-budget) → taken (checklist).
- **Eigenaarschap:** elk project heeft een **rol** (eigenaar) + een **trekker** (persoon).
  Uren per project = het budget; de rol logt de **werkelijke uren** bij het afronden.

## Werkwijze uren
Uren per project = budget (deze lijst). De rol vult bij het afronden van een project (of taak)
de werkelijke uren in. Die rollen op naar werkpakket → objective. Export = eigen urenoverzicht +
PDF. De urenstaat wordt zo een afgeleide, niet een tweede administratie.
