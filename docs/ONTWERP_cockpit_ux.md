# Ontwerp — cockpit UX (na review-1)

De cockpit is een SIGNAAL, geen datadump ("it takes a village to raise a Noochie"). Kernklacht uit
review-1: te veel verschillende patronen, informatie verspreid. Daarom: alles valt terug op vier
canonieke patronen. Toets elk blok hieraan.

## De vier patronen
1. **Werk-in-focus = de enige beslis-flow.** Alles dat jouw oordeel vraagt gaat via /triage
   (werk-in-focus) of wordt direct een project. Geen losse beslis-knoppen verspreid.
2. **Lijst = top 3 + toggle + "naar volledige lijst".** Elke lijst toont standaard top 3,
   in/uitklapbaar, met altijd een doorklik naar het geheel.
3. **Harmonica voor kennis.** Links de lijst, rechts het opengeklapte detail; tab-doorloop;
   plek voor losse (niet-officiële) notities.
4. **Chat-interface voor reflectie en projecten.**

## Brokken (volgorde)
1. **Opschoning + hero** (deze brok): cockpit rustig maken.
   - Weg: tellingen-balk, BHAG uit hero, weekrapport, inbox-blok, dubbele Noochie-blok onderaan.
   - Verplaatsen: core values → huisregels; target + bezoekers + conversie → Website Watcher.
   - Hero: missie direct onder de titel; "aan jou"-kaart krijgt het aantal lopende projecten;
     "het dorp werkt voor je" wordt Noochie-reflectie + link naar het dagbulletin; daaronder
     direct de eerste spanning om te verwerken (CTA naar werk-in-focus).
2. Lijst-pattern overal (top 3 + toggle + volledige lijst).
3. Harmonica voor de kennislaag (inzichten/onbeslist/gaten/voorgestelde links samengevoegd).
4. Stuur-paneel: kwartaaldoel + Focus op (max 3) + Focus niet op (max 3) — bewerkbaar.
5. Chat-interface (Noochie-reflectie + projectgesprekken).

## Losse fixes (meenemen waar logisch)
- Projectstatus op de knop tonen (niet naar boven springen).
- "Maak content-project" → automatisch naar projectbord copywriter.
- Website Watcher: bezoekers + resultaat t.o.v. doel + conversie (orders/visitors).
- Geschreven perspectief (5 regels, Harry/Scout) bij inzichten en volg-woorden.
- Concurrenten "nieuw gespot" → in de gemonitorde lijst, locked, met toggle.
